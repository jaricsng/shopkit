## What's here

- `security/` — OWASP manual checks + ZAP scan wrapper
- `observability/` — Jaeger + Prometheus + Grafana overlay
- `load-testing/` — k6 + Locust scenarios
- `iac-terraform/gcp-cloud-run/` — parameterized Cloud Run module, `app_name` pre-set to `shopkit`
- `governance/policy-as-code/` — example Conftest/OPA policy-as-code guardrails for the Terraform module
- `.claude/commands/` — Claude Code review/fix commands
- `.github/workflows/`, `.pre-commit-config.yaml` — CI/CD pipeline shape + pre-commit security baseline (incl. IaC + migration checks)
- `.github/workflows/drift-detection.yml` — scheduled `terraform plan` drift check (ships gated; configure cloud auth to enable)
- `.gitignore`, `.secrets.baseline` — `.env` can't be committed; detect-secrets hook works on first commit
- `Makefile`, `.devcontainer/`, `.tool-versions`, `.env.example` — paved inner loop (`make help` for the task list)
- `operations/` — rollback / incident / postmortem runbooks + SLO definitions
- `tools/check_migrations.py`, `docs/DATABASE-MIGRATIONS.md`, `docs/FEATURE-FLAGS.md` — schema-safety gate + safe-release patterns
- `catalog-info.yaml` — Backstage catalog registration (fill in the owner/project-slug TODOs)
- `.github/CODEOWNERS`, `.github/dependabot.yml` — review ownership + dependency-update policy (fill in the CODEOWNERS TODO)
